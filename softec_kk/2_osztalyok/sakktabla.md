# Sakktábla

(+/-) Csináljatok sakktáblát. Azaz csak minden második négyzetet jelenítsétek meg.

> [!Tip]
> A `%` (modulo) operátor az osztási maradékot képzi. Az alábbi kódrészlet eldönti egy számról, hogy páros-e:

``` csharp
if (a % 2 == 0)
{
	//páros
}
else
{
	//páratlan
}
```
